# Game Space Server

**Easily manage your game folders and move them between disks!**

---

## Overview

Game Space Server helps you free up space on your main disk (NVMe/SSD) by moving games to a secondary disk (HDD), and restoring them when needed.
No more uninstalling and reinstalling large games just to play them again!

- **Supported OS:** Windows & Linux

---

## Installation

1. Download the latest release from the Release Page
2. Extract the zip file to a folder of your choice
3. Run `GameSpaceServer.exe`

## Known Issues

- If you move a folder to a disk without enough space, the operation may fail and leave the folder in an inconsistent state

---

## Upcoming Features

*Please consider donating to support development of these features!*

- Check for enough space on the target disk before moving folders
- Request the "last accessed" game folder to be moved
- Automatically move multiple folders based on last accessed time and user-defined "required space"
- Archive mode: leave folder structure intact, replace executables with a single restore executable (automatic Steam restore)
- Automatic background task to move last accessed game folder based on user-defined "minimum free space"
